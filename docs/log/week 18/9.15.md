##### Java使用正则表达式简单校验手机号码

```java
String phone = "18115178805";
String regex = "^[1]\\d{10}$"; // 1开头的11位数字
Pattern pattern = Pattern.compile(regex);
boolean matches = pattern.matcher(phone).matches();
if (!matches) {
    System.out.println("手机号格式不正确");
}
```
