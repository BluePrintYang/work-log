### cmd查看进程

1、查看端口占用

```bash
netstat -ano
```

2.查看指定端口的占用情况

```bash
netstat -aon|findstr 8080    -- 数字为端口号
```

3.查看PID对应的进程

```bash
tasklist|findstr 5769  -- 数字为PID
```

4.结束该进程

```bash
taskkill /f /t /im xxx.exe -- 杀程序
taskkill /f /PID 5769  -- 杀PID
```
